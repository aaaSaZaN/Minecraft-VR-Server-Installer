# Minecraft VR Server Installer

Автоматический установщик нативного сервера Minecraft с поддержкой VR (Vivecraft) и голосового чата (Simple Voice Chat) для Windows.

## 🚀 Быстрая установка в 1 клик

Для установки сервера просто откройте **PowerShell** (нажмите `Win + X` и выберите **PowerShell** или **Терминал**) и запустите следующую команду:

```powershell
irm https://raw.githubusercontent.com/aaaSaZaN/Minecraft-VR-Server-Installer/master/setup.ps1 | iex
```

---

## 🛠 Что делает этот скрипт?

1. **Выбор параметров**: Спросит версию игры (рекомендуется `1.20.1`), тип ядра (`Forge`, `Fabric` или `NeoForge`), объем выделяемой оперативной памяти и порт сервера (по умолчанию `25565`).
2. **Установка Java**: Автоматически определяет и скачивает нужную портативную версию Java (Eclipse Temurin) под капот сервера (вашему компьютеру это не навредит).
3. **Установка ядра**: Скачивает и разворачивает выбранную версию ядра Minecraft.
4. **Скачивание модов**: Устанавливает серверную часть модов **Vivecraft (VR)** и **Simple Voice Chat** под выбранную версию.
5. **Создание панели управления**: Размещает на вашем Рабочем столе ярлык `Minecraft_Server_Control.bat`.

---

## 🖥 Панель управления (Ярлык на рабочем столе)

Запустив созданный ярлык, вы сможете:
1. **Запустить сервер**: Сервер откроется в отдельном черном окне. Вы можете играть!
2. **Остановить сервер**: Краткая инструкция по безопасному выключению (ввести команду `stop` в окне сервера).
3. **Выдать админку (OP)**: Краткая инструкция по выдаче прав администратора (ввести команду `op Игрок` в окне сервера).
4. **Поменять настройки**: Заново запускает установщик, если вы решили изменить оперативную память или моды.
5. **Открыть папку с модами**: Быстрый переход в папку `mods` сервера (чтобы добавить свои моды вручную).
6. **Открыть папку сервера**: Открывает корневую папку сервера со всеми файлами.

---

## 📌 Как играть?

* В главном меню панели управления будет показан ваш **IP для подключения** в локальной сети (например, `192.168.1.70:25565`).
* Передайте этот IP друзьям, которые находятся в вашей домашней сети (или подключены через Hamachi / Radmin VPN), чтобы они могли войти в игру.
